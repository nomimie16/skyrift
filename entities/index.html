Afin de remporter la victoire, le jeu vous permet de sélectionner parmi plusieurs types d’unités.

Les dragons:

Dragonnets Dragon moyen Dragon Géant

  • Les dragonnets

    Des dragons petits et rapides.

    • Forces : rapide, combat rapproché,

    • Faiblesses : peu résistant, petites flammes.

  • Les dragons moyen

    Les dragons de taille moyenne.

    • Forces : Robuste, difficile à blesser.

    • Faiblesses : Déplacements.

  • Les dragons à deux têtes

    Les plus puissants des dragons.

    • Forces : Tirs renforcés, extrêmement résistants.

    • Faiblesses : Taille massive, lents.

Les tours :

Tour de défense

  • Les tours de défenses

    • Forces : un rempart puissant pour protéger votre base contre les assauts ennemis.

    • Faiblesses : statiques et ne peut être placée que devant votre base afin de la protéger.
