## Dépendances principales

### Python
* [Python 3.x](https://www.python.org/downloads/) 
* Version souhaitée : 3.10 ou supérieur
* Fournit l’environnement d’exécution principal du projet.

### Pygame
* [Pygame](https://www.pygame.org/news) 
* Version souhaitée : ≥ 2.5 
* Bibliothèque utilisée pour l’affichage, les sprites, les événements et le son.

### Pygame GUI
* [Pygame GUI](https://pygame-gui.readthedocs.io/en/latest/) 
* Version souhaitée : ≥ 0.6.9
* Framework d’interface graphique basé sur Pygame (boutons, panneaux, menus, thèmes).

### Numpy
* [Numpy](https://numpy.org/) (maths/IA)
* Version souhaitée : ≥ 1.23
* Outils mathématiques avancés, manipulation de matrices et algorithmes IA.

### Moviepy
* [Moviepy](https://zulko.github.io/moviepy/) 
* Version souhaitée : ≥ 1.0.3
* Génération et montage de clips vidéo (montage, export, effets).

### Pytest
* [Pytest](https://docs.pytest.org/en/stable/) 
* Version souhaitée : ≥ 7.0
* Framework de tests automatisés pour valider les classes du projet.

## Installation des dépendances

* Pour installer directement l'ensemble des dépedances requises, ouvrez le dossier contenant le projet dans un terminal et entrez cette commande :

``` 
pip install -r requirements.txt
```
