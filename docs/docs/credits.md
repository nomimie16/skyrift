## Assets

!!! warning "Information : "
    L'ensemble des **sprites** et des éléments d'interface ont été réalisés en **pixel art** à l'aide du logiciel [Pixelorama](https://pixelorama.org/).
---

### Sprites d'unités

Les **sprites** situés dans `assets/sprites/` regroupent :

* Les dragons

!!! info "Information : "
    Chaque dragon dispose d’une sprite sheet avec 5 états différents correspondant à ses animations.
---


### Éléments d'interface

Les composants de **l’interface utilisateur** stockés dans `assets/` contiennent :

* Les boutons
* Les barres de vie
* Les éléments de la boutique

---


### Environnement

Les **éléments de l'environnement** stockés dans `assets/` reprennent :

* Les bases
* Les bourses
* Les îles, volcans et tornades

!!! info "Information : "
    Chaque élément possède un sprite unique sauf la tornade qui est animée.

## Contact et support

### Communication

* GitHub Issues : **Rapports** de bugs et suggestions
* Discord : Canal de de **disscussion**
* Contributions : Ajout de nouvelles **fonctionnalités**


### Contributions

Vous pouvez **contribuer** au projet en y ajoutant de nouvelles fonctionnalités. Pour contribuer suivez ces instructions :

1. **Forkez** le repository GitHub
2. Créez d'une branche pour vos modifications
3. **Testez** et **validez** vos changements
4. Faites une **pull request** avec description détaillée
