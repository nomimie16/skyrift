## Glossaire 

| Terme        | Définition                                     |
|--------------|------------------------------------------------|
| Entité       | Élément logique du jeu (dragon, ennemi, objet) |
| Zone d’effet | Zone appliquant un effet temporaire            |
| Sprite       | Elément graphique qui peut se déplacer sur l'écran         |
| SpriteSheet  | Série de sprites, servant à créer des animation        |



## Nommage des fichiers

* Classes : `Dragon`
* Fichiers : `dragon.py`
* Constantes : `DRAGONNET_COST`
* Fonctions : `move_dragon`

## Commandes utiles

**Lancer le jeu :**

```
python main.py
```

**Lancer les tests :**

```
python -m unittest
```

**Télécharger les dépendances :**

```
pip install -r requirements.txt
```
